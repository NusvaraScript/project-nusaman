Isi ini dibuat untuk tugas praktek PHP Native: situs statis + panel admin CRUD layanan.

Instruksi singkat:

- Import `db.sql` ke MySQL (mis. melalui phpMyAdmin) untuk membuat database `nusantara_db` dan tabel `layanan`.
- Sesuaikan kredensial DB di `admin/inc/db.php` jika diperlukan.
- Jalankan XAMPP, lalu buka `http://localhost/projek-nusaman/home/home.html`.
- Panel admin: `http://localhost/projek-nusaman/admin/login.php` (username: `admin`, password: `password`).

Catatan: implementasi login di contoh ini memakai kredensial statis untuk kemudahan ujian. Untuk produksi gunakan tabel users dan password hashing.
